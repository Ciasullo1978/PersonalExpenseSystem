"# Personal Expense System" 
README
##  Requisiti
* **MySQL Server** installato e attivo.
* **Python 3.8+**
* **Libreria:** `mysql-connector-python`

##  Installazione Dipendenze
Esegui il comando:
```bash
pip install mysql-connector-python

Configurazione
1.	Importa lo script sql/database.sql nel tuo server MySQL.
2.	Modifica il dizionario db_config in src/main.py con le tue credenziali.
3.	Avvia l'app:
Bash
python src/main.py
